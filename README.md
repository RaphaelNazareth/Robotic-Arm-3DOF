3DOF Robotic Arm with HyperPack Optimization
This project features a 3DOF robotic arm that uses a magnet to efficiently pick and place items. The system integrates computer vision to detect object coordinates and employs inverse kinematics for precise movement. To optimize item placement, the HyperPack algorithm is implemented, enhancing spatial efficiency through intelligent packing strategies.
